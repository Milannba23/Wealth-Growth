## üßÆ Calculation Logic

The application uses the following financial formula:

$$A = P(1 + r)^n + PMT \times \frac{(1 + r)^n - 1}{r}$$

*Where: A = Final Amount, P = Principal, r = Interest Rate, n = Number of periods, PMT = Monthly Payment.*
